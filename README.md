## Prerequisites

- Docker + Docker Compose

## Build & Run

Check out repo:

```bash
git clone https://github.com/607011/redact.git
cd redact
```

Build and run Docker images:

```bash
docker compose --profile dev up --build
```

Open [127.0.0.1:8880](http://127.0.0.1:8880/) in your favorite browser.
